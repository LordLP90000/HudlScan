C. Migration Guard (Copy Exactly)

Copy Rules For Any AI Migration Agent

Rule 1: Keep page flow and conditional states exactly as shown in section B.
Rule 2: Do not rename visible button labels unless explicitly requested.
Rule 3: Do not change endpoint path or payload contract during TS migration.
Rule 4: Convert implementation language and tooling only, preserve UX behavior.
