## File Requirements

### ✅ Acceptable Files:

**Images:**
- PNG files (.png)
- JPEG files (.jpg, .jpeg)
- WebP files (.webp)

**Documents:**
- PDF files (.pdf)

**Size:**
- Up to 10MB per file
- Can upload multiple files at once

**Quality:**
- Resolution: 1000px or higher recommended
- Text must be readable
- Good contrast/lighting

### ❌ NOT Acceptable:

- ❌ Word documents (.doc, .docx)
- ❌ PowerPoint files (.ppt, .pptx)
- ❌ Excel files (.xls, .xlsx) - output format, not input
- ❌ Video files
- ❌ Blurry or low-resolution images
- ❌ Handwritten notes (unless very clear)

## Tips for Best Results

### 📸 Screenshot Tips:
1. Use screenshot tool (not phone camera of screen)
2. Capture full play including title
3. Include all text/labels
4. Don't crop too tightly
5. One play per screenshot is best

### 📄 PDF Tips:
1. Export directly from Hudl if possible
2. Don't scan paper playbooks (use digital export)
3. Keep file size reasonable (<50MB)
4. One section at a time works best
5. Text-based PDFs work better than image PDFs

### 🎯 General Tips:
- Upload in batches (10-15 files at a time)
- Always review AI extractions
- Add personal notes in the 4th column
- Save the Excel file with game/date in filename
- Keep original images as backup

## Common Scenarios

### Scenario 1: "I have 50 plays to convert"
**Solution**: Upload 10-15 at a time, review each batch, combine Excel files later

### Scenario 2: "My playbook has diagrams and text"
**Solution**: AI extracts text; diagrams help context but aren't in final output

### Scenario 3: "I need multiple versions (offense, defense, special teams)"
**Solution**: Process each separately, export to different Excel files

### Scenario 4: "My plays have complex blocking schemes"
**Solution**: AI extracts main points; add details manually in editor
